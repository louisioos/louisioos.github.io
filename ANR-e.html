ANR Project QCM                                   

Title : Quantization of Character varieties as a Model for quantum chaos.

Description : The QCM project is an ANR project "Jeune Chercheur, Jeune Chercheuse" (JCJC), which will be running from October 2023 to September 2027.
The official project denomination is ANR-23-CE40-0021-01.

Permanent members : The ANR project counts seven permanent members :
  • Louis Ioos (CY Cergy, Project Coordinator)
  • Renaud Detcherry (Dijon)
  • Ramanujan Santharoubane (Paris-Saclay)
  • Cristina Anghel (Geneva)
  • Siarhei Finski (Polytechnique)
  • Martin Puchol (Paris-Saclay)
  • Alix Deleporte (Paris-Saclay)

Positions : The ANR project has a postdoc position at the department AGM of CY Cergy Paris University under the directions of the project coordinator, which will be held by Wei-Chuan Shen from September 2024.
